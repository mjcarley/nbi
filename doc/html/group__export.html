gdouble *  dat,
gint  dstr,
gint  ne,
gint  nd 
)

write numerical data to file.

The function writes a header which can be interpreted by nbi_data_read and writes nd lines of data in the form:

dat[i*dstr+0] dat[i*dstr+1] ... dat[i*dstr+ne-1]

with .

Parameters
foutput file stream;
datdata;
dstrstride between first components of successive data entries;
nenumber of components per data entry;
ndnumber of data entries.
Returns
0 on success.
