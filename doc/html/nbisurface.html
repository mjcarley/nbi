Generating surfaces for NBI calculations

nbi-surface generates and discretizes surface geometries and outputs them in the NBI file format.

nbi-surface -h

gives a current list of command line options. The most important of these are:

  • -q number of quadrature points per surface patch;
  • -o output file name;
  • -g built-in surface geometry;
  • -G list available built-in geometries;
  • -m GMSH geometry file name;
  • -a AGG geometry file name;
  • -i add an integer argument to pass to the built-in geometry generator;
  • -f add a floating point argument to pass to the built-in geometry generator.

Option -f, -m, or -a must be specified.
