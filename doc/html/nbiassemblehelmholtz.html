Assembling matrices for Helmholtz problems

nbi-assemble-helmholtz takes an input geometry, generated by nbi-surface (Generating surfaces for NBI calculations) and outputs the system matrices required to solve the boundary integral problem using nbi-solve-helmholtz (Solving Helmholtz problems)

nbi-assemble-helmholtz -h

gives a current list of command line options. The most important options are:

  • -k the wavenumber in the Helmholtz equation (this must be set);
  • -g the geometry file name (this must be set);
  • -u the number of upsampled nodes per patch (default is 25);
  • -n separation parameter for patch neigbourhoods (default 1.25);
  • -m file name for matrix data on output.
